# P2 Source And Concept Verification

## Accepted
- Thinking Machines supports the two-layer idea conceptually, but LearningAI
  must call it app-native continuous orchestration, not native interaction-model
  training.
- Fine-tuning is not the primary adaptive learner-state strategy. Use structured
  state, retrieval, summaries, instructions, and evals first.
- BKT is the first auditable mastery baseline.
- DKT, AKT, LPKT, language-model LKT, CIKT, LLM-KT, and RAG-KT stay research or
  benchmark tracks until LearningAI data proves value.
- AWS/Dexie split is valid if each store has a scoped responsibility.
- Background mastery updates are high risk and require evidence, validation,
  audit rows, and rollback.
