# P2 Learning Scientist

## Objective
Argue for the safest adaptive learning architecture: KT, evals, evidence ledger, guardrails, and measurable pedagogy.

## Ideology
A smart tutor must not hallucinate mastery. LLMs may explain, extract, and generate; evidence-based learner modeling should update mastery.

## Required Sources
- BKT, Logistic KT, DKT, Language-model KT / LLM-KT / CIKT / RAG-KT as relevant.
- OpenAI evals and structured outputs docs where useful.
- Learning science sources where useful.

## Expected Output
- Argument with citations.
- What the book must tighten.
- What should be beta source of truth vs research track.
- Rebuttal points expected from other ideologies.
