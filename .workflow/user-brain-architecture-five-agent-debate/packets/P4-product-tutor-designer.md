# P4 Product Tutor Designer

## Objective
Argue for the natural human tutor experience: voice timing, tool presentation, images, code snippets, charts, and reader flow.

## Ideology
The learner should experience one calm tutor, not an architecture diagram. Multimodal artifacts should appear at the right teaching moment.

## Required Sources
- Thinking Machines interaction models.
- OpenAI Realtime/voice/tool docs.
- Sources on multimodal tutoring or learning design where useful.

## Expected Output
- Argument with citations.
- UI/reader improvements and diagram/image recommendations.
- Concrete tool/artifact behaviors.
- Rebuttal points expected from other ideologies.
