# Packet P1: Cognitive Modeling Fit

## Objective
Evaluate whether LearningAI's User Brain Architecture matches how learner
knowledge, misconceptions, retrieval practice, metacognition, and transfer
actually evolve.

## Source Focus
- Academic papers and classic intelligent tutoring research.
- Cognitive modeling, BKT roots, ACT-R-adjacent evidence, retrieval practice,
  cognitive load, spacing, misconception repair, and transfer.

## Expected Output
- Strong points in the current architecture.
- Cognitive-modeling mismatches or overclaims.
- Source-backed recommendations.
- Claims to reject or mark research-track only.
- Debate-ready summary with URLs/citations.

## Rules
- Use primary papers where possible.
- Do not assume pause, confidence, or fluency proves mastery.
- Nested subagents: use at most one if available; otherwise split notes into
  academic, implementation, and product-risk subsections.
