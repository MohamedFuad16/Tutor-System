# Packet P4: Long-Term Memory Formation

## Objective
Evaluate how LearningAI should form durable learner memory over weeks/months:
episodic, semantic, procedural, reflective, and preference memory.

## Source Focus
- Memory consolidation, summarization, forgetting, retrieval, spaced review,
  and learner-model auditability.
- AI memory systems, RAG memory, graph memory, and open-source agent memory
  implementations.
- Privacy and retention practices for personal learning records.

## Expected Output
- Memory taxonomy for LearningAI.
- What to store, summarize, forget, retrieve, and audit.
- Risks around privacy, stale memories, and false personalization.
- Concrete architecture recommendations.
- Debate-ready summary with URLs/citations.

## Rules
- Separate learner facts from tutor guesses.
- Make memory reversible and source-linked.
- Nested subagents: use at most one if available; otherwise split notes into
  cognitive memory, AI memory systems, and privacy/audit subsections.
